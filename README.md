# IAforDev

Uma trilha completa e prática para formar Desenvolvedores de Inteligência Artificial, cobrindo desde os fundamentos matemáticos e programação em Python até LLMs, RAG, Agentes, Cloud AI e MLOps.

---

## Objetivo

Capacitar desenvolvedores a projetar, construir, implantar e monitorar soluções modernas de Inteligência Artificial utilizando ferramentas e frameworks amplamente adotados pelo mercado.

## Público-Alvo

- Desenvolvedores de Software
- Engenheiros de Dados
- Cientistas de Dados iniciantes
- Estudantes de Computação
- Profissionais de Tecnologia interessados em IA
